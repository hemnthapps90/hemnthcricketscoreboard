# 🏏 hemnthcricketscoreboard – Live Cricket Scraper Dashboard

hemnthcricketscoreboard is a **Python-based live cricket scraping tool** that collects real-time match data and displays it in a **local web dashboard**.

---

# ⚙️ Requirements

Before running the project make sure the following tools are installed:

* Python 3.8+
* Git
* Google Chrome

---

# 🔍 Check if Python is Installed

### Windows

Open **Command Prompt** and run:

```
python --version
```

If Python is installed you will see something like:

```
Python 3.10.5
```

If not installed, download it from:

https://www.python.org/downloads/

During installation **check the option:**

```
Add Python to PATH
```

---

### Mac

Open **Terminal** and run:

```
python3 --version
```

If Python is not installed, install it using **Homebrew**:

```
brew install python
```

Or download from:

https://www.python.org/downloads/

---

# 🔍 Check if Git is Installed

### Windows / Mac

Run:

```
git --version
```

If Git is installed you will see:

```
git version 2.x.x
```

If Git is not installed:

Download from:

https://git-scm.com/downloads

Install using default settings.

---

# 📥 Clone the Repository

Once Git is installed run:

```
git clone https://github.com/hemnthapps90/hemnthcricketscoreboard.git
cd hemnthcricketscoreboard
```

---

# 📦 Install Dependencies

### Mac / Linux

```
pip3 install -r requirements.txt
```

### Windows

```
pip install -r requirements.txt
```

If `pip` does not work:

```
python -m pip install -r requirements.txt
```

---

# ▶️ Run the Application

### Mac

```
python3 main.py
```

### Windows

```
python main.py
```

---

# 🔗 Enter Match Link

Enter a live match link that ends with **live**.

Example:

```
https://crex.com/cricket-match/xxxx/live
```

---

# 🌐 Dashboard

The app will automatically open:

```
http://localhost:8000
```

Here you will see:

* Live Score
* Batsmen Stats
* Bowler Stats
* Commentary
* Win Predictor
* Playing XI

Data updates automatically every **15 seconds**.
